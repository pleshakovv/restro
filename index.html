Пакетні пропозиції

Standart

Що входить до пакету Standart:

  1. Абонплата 10 євро в місяць
  2. Готовий вітальний бот в месенджері Fb ( далі самостійне оновлення інформації )
  3. Бот для персоналу базовий “Обмежений” ( до 300 карток + кожна додаткова картка )
  4. Термін виконання завдання 5 робочих днів ( при умові надання всієї необхідної інформації для наповнення бота )
$199

Business

Що входить до пакету Business:

  1. Абонплата в місяць 15 євро + 3 перші місяці безкоштовні
  2. Підтримка в Telegram каналі
    - Оновлення та редагування інформації, акцій, калькуляцій, тощо ( в робочий час на протязі години, у вихідні дні на протязі дня )
    - Push - повідомлення в боті для інформування персоналу
    - Створення та постійне оновлення вітального бота в мессенджерах FB, Telegram, Viber для клієнтів ( Акції, Новинки, тощо )
    – Push - повідомлення в боті для інформування клієнтів
  3. Бот для персоналу “Необмежений”
  4. Термін виконання завдання 5 робочих днів, у разі якщо кількість карток не перевищує 300 позицій.
$299
